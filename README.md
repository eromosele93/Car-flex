Car-Flex is a modern single-page application that allows users to list their cars for rent or sale, manage offers, and interact through a seamless, real-time interface. Built with Laravel, Vue.js, and Tailwind CSS, the platform provides a smooth and responsive user experience powered by a secure MySQL backend.

Users can create an account, upload their vehicle details, and receive multiple purchase offers, choosing the best one. The system also supports car rentals, enabling users to rent vehicles for a selected period and make secure payments through Stripe. Integrated in-app notifications keep users updated on new offers, rental confirmations, and other activities.

Car-Flex showcases full-stack development with dynamic UI components, real-time interactions, secure payments, and a scalable backend. delivering an efficient and intuitive car marketplace experience.
